# GitHub Configs - Agent Instructions

GitHub-asetukset ja workflowt.

## Saatavilla olevat tiedostot

| Tiedosto | Sisältö |
|----------|---------|
| `branch-protection.md` | Branch-suojausasetukset |
| `actions-templates/` | GitHub Actions workflow-pohjat |

## Käyttö

Uudessa projektissa:
1. Konfiguroi branch-suojaus `branch-protection.md` mukaan
2. Kopioi tarvittavat Actions workflowt

## Huomioitavaa

- **Branch-suojaus** estää suoran pushin mainiin
- **Actions** voi korvata Vercelin automaattideploymentin jos tarvitaan
